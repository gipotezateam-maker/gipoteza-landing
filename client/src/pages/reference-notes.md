# Референс: Дизайн игры ЗАПУСК

## Топ-бар (header)
- Левый угол: логотип "ЗАПУСК" (жирный, белый) + подпись "Маркетинговый симулятор для EdTech" (мелкий серый)
- Метрики в ряд: иконка + название + число + дельта + мини-спарклайн
  - Пользователи: 12 580 (-8%) — красный спарклайн
  - Выручка: 8,4 млн ₽ (+12%) — зелёный спарклайн  
  - NPS: 32 (-10) — красный спарклайн
  - Популярность: 68 (+5) — зелёный спарклайн
  - Неделя: 12/24 — иконка календаря
- Правый угол: кнопка-меню (три горизонтальные линии, тёмный круг)
- Фон топ-бара: тёмный #1a1a1a, нижняя граница тонкая

## Основной экран (3 колонки)
### Левая колонка (~220px)
- Карточка "Ситуация" с красным заголовком + иконка 🔥 + "Горячая"
- Текст ситуации (белый, мелкий)
- Карточка "Цель" с иконкой мишени, красный текст цели + прогресс-бар красный

### Центр (широкий, ~60%)
- Фоновое изображение: CEO и CMO на тёмном фоне с боке-эффектом (офисная атмосфера)
- CEO слева, CMO справа
- Речевой пузырь CEO: тёмный фон, красный заголовок "CEO", крупный белый текст реплики
- Под CEO: маленькая карточка "CEO | ⚡ Хочет быстрых результатов"
- Под CMO: маленькая карточка "СМО (Вы) | 🧠 Думает на 3 шага вперёд"

### Нижняя часть (карточки решений)
- Заголовок "Какой шаг вы сделаете?" — белый, по центру
- 4 карточки в ряд 2+2:
  - Карточка 1 (красная): номер "1", иконка 🔥, заголовок "Снизить цены", описание
  - Карточка 2 (жёлтая): номер "2", иконка 📢, заголовок "Усилить рекламу", описание
  - Карточка 3 (зелёная): номер "3", иконка 😎, заголовок "Сделать акцент на ценности", описание
  - Карточка 4 (фиолетовая): номер "4", иконка 🎁, заголовок "Добавить бонусы", описание
  - Каждая карточка: стрелка → в правом нижнем углу

### Нижняя панель
- Левый угол: кнопка "📖 Журнал"
- Правый угол: кнопка "💡 Подсказка 3"

## Цвета карточек решений
1. Красная: bg rgba(180,30,20,0.7), border красный
2. Жёлтая/оранжевая: bg rgba(160,100,0,0.7), border жёлтый
3. Зелёная: bg rgba(20,100,50,0.7), border зелёный
4. Фиолетовая: bg rgba(80,30,120,0.7), border фиолетовый

## Ключевые особенности
- Нет отдельного интро-экрана с "VS" — сразу игровой экран
- Персонажи занимают всю высоту центральной части
- Речевой пузырь CEO — крупный, доминирует
- Карточки решений — плотные, с большими иконками
- Общий тон: тёмный, тёплый, кинематографический
